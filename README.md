# Vijay College of Education Website

Pure HTML + CSS + JavaScript multi-page website recreated from the supplied reference screenshots.

## Pages
- index.html
- about.html
- courses.html
- admission.html
- facilities.html
- placement.html
- contact.html

## Run
Open `index.html` directly, or use VS Code Live Server.

Note: Demo photography uses remote Unsplash image URLs, so internet access is required for those images to load.
