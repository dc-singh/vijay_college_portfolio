# Vijay College of Education Website

This is a public-facing multi-page website for Vijay College of Education. It presents the college profile, academic programs, admission process, campus facilities, placement support, and contact details in a simple HTML, CSS, and JavaScript format.

## Public Information

Visitors can learn about:

- The college overview and mission
- Academic programs such as B.Ed., D.El.Ed., B.El.Ed., and B.A. B.Ed.
- Admission steps, eligibility notes, and important dates
- Campus facilities including the library, computer labs, smart classrooms, and sports areas
- Placement support, training, and recruiter information
- Contact details for admissions and general enquiries

## Notes

- The website uses remote Unsplash image URLs for demo photography, so internet access is required for those images to load.
- The form interactions are frontend-only and currently show a confirmation message instead of sending data to a server.
